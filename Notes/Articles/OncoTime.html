The OncoTime Programming Language

Introduction

The OncoTime Programming language was designed by professor Laurie Hendren as a solution to problems faced by doctors in the Radiation Oncology laboratory at the Montreal General Hospital. The hospital had a program, wherein Doctor, Patient, and Diagnosis data was all stored in a MySQl database. The issue was that Doctors, not trained in programming or database querying, found it very difficult to access the data and information they needed. OncoTime acted as a solution.

OncoTime is a general purpose, "wrapper" language that allows users to quickly write simply programs that will allow them access to the information they need, quickly, and efficiently. It was written in the C Programming Language using Flex / Bison My team developed the otc OncoTime compiler which took OncoTime programs and created Python code that would access and display the results of the MySQL queries to the user.

We are currently in the process of making the source code available unders and open source license. The idea being that we want everyone to have access to the OncoTime language, as well as allow intelligent developers to help us further develop the language.
